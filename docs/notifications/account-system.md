# 账号迁移说明

UsagiPass 与 兔卡账号 已完成深度融合升级，现统一更名为 UsagiLab 账号。

1. 原 UsagiCard 账号直接变为 UsagiLab 账号，可以直接使用原用户名和密码登录。上传过的图片将被正常保留。
2. 原 UsagiPass 账号也会变成 UsagiLab 账号，但是初次只能使用水鱼或者落雪登录，第一次登录后，将要求您重新设置用户名、邮箱和密码。设置完成后，可以通过三种方式登录（用户名密码、水鱼、落雪）。上传过的图片、个性化设置、绑定的查分器账号将被正常保留。
3. 考虑到部分用户可能同时使用过这两个平台，为了防止多账号带来的不便，我们添加了账号合并功能。任意登录一个账号后，进入设置，点击头像后方的三个点进入菜单，选择账号合并即可。

> 关于账号合并的更多信息，请进入账号合并页面查看详细说明。
