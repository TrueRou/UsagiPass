# 账号迁移说明

UsagiPass 与 兔卡账号 已完成深度融合升级，现统一更名为 UsagiLab 账号。

您的账号将发生以下变化：

1. 原 UsagiCard 账号直接变为 UsagiLab 账号。可以使用账号和密码登录。
2. 原 UsagiPass 账号也会变成 UsagiLab 账号，但首次只能使用第三方登录。
3. 原 UsagiPass 账号第三方登录后，将要求完善账号信息，完善后即可使用任意方式登录。

您的数据将发生以下变化：

1. 原有图片、个性化设置将保留。
2. 原有的落雪查分器 Personal-Token 将保留，但水鱼查分器 Import-Token 需要重新绑定。

如果您同时拥有 UsagiPass 和 兔卡账号：

- 我们添加了账号合并功能。任意登录一个账号后，进入设置，点击头像后方的三个点进入菜单，选择账号合并即可。
- 关于账号合并的更多信息，请进入账号合并页面查看详细说明。
